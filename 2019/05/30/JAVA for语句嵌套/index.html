JAVA for语句

使用java中的for语句实现一些功能，这里简单介绍for语句的一些使用方法以及使用语法格式。

for循环语句的语法格式

for的语法格式

for关键字,循环
循环: 指同一个逻辑,需要多次执行,使用循环
举例: 登录界面,客户姓名,客户密码,姓名和密码都正确,登录成功
如果姓名或密码输入错误了,给你三次机会,这三次机会就可以使用循环来完成这个功能

说明

1.初始化值 int i = 0;
2.判断语句,必须是boolean类型的逻辑,i < 10
3.变量值的变化i++
for(int i = 0; i < 10;i++){// 功能: 能将0-9的整数打印出来
System.out.println(i);
}
4.只要判断语句为true,那么for循环中的语句就会执行。
